# Como instalar o projeto UnBomPlanner:

OBS: O tutorial abaixo funciona para ambientes UNIX ou MacOS.
OBS2: Para ambientes Windows, o seguinte tutorial pode apoiar na instalação do Node, NPM e React:
 https://www.devmedia.com.br/como-instalar-o-node-js-npm-e-o-react-no-windows/40329

1 - Instale o NodeJS na sua máquina caso ainda não tenha (recomenda-se a versão LTS mais nova):
> https://nodejs.org/en/

1.1 - Verifique que o Node está instalado com o comando:
> node -v

1.2 - Descompacte o arquivo .zip com o código do projeto

2 - Em um terminal de linha de comando, navegue até a pasta do projeto:
> cd UnBomPlanner-main

3 - Instale as dependências do projeto com o seguinte comando:
> yarn install

4 - Inicialize o projeto com o seguinte comando:
> yarn start
